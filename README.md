# CONTROLE FINANCEIRO

## Sobre
Aplicação Web para controle de receitas e despesas, permitindo adicionar transações organizadas por categorias e visualizar saldo total.

## Tecnologias
JavaScript, HTML, CSS, LocalStorage

## Funcionalidades
- Adicionar transações
- Remover transações
- Calcular saldo automaticamente
- Separar receitas e despesas
- Salvar dados no navegador
